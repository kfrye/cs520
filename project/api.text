Create Graphics API

Import the module:
import “createGraphics” as createGraphics

Create the graphics object:
var graphics := createGraphics(width, height)

Add shapes:
var circle := graphics.addCircleAt(x@y)ofRadius(length)
circle.color := “red”
circle.fill := true

Add click handler:
circle.click := block

Draw objects:
circle.draw (draws one object)
graphics.draw (draws all objects)
